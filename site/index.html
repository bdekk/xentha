Welcome to the new local gaming platform. Xentha tries to become an open source initiative to couch gaming. Look at our games section to find just the games you want to play right now.

Open Source

We try to be as open as possible. All our code is on github. Next to that, we have a slack channel where developers are waiting for you to give you help in creating your own games.

Developers wanted!

Would you also like to develop a game? Or would you rather help us improve our API and support all those game developers out there? Feel free to browse through our developer documentation and API documentation.
